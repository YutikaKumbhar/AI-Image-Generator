📌 Project Overview
This project is an AI-powered Image Generator that converts user text prompts into realistic or artistic images using Generative AI models.

Users can enter a description (prompt), choose a style, and generate images instantly. The system leverages modern AI techniques to create high-quality visuals based on natural language input.

Features:
Text-to-Image generation
Multiple style options (realistic, anime, artistic, etc.)
Fast image generation
User-friendly interface
Option to download generated images
